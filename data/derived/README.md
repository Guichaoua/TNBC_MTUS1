Derived tables included for transparency and manuscript reproduction.

- `source_tnbc_core_gsea_nes.csv`: normalised enrichment scores used for the TNBC pathway panel.
- `source_tnbc_core_gsea_fdr.csv`: adjusted p-values used for the TNBC pathway panel.
- `source_depmap_all_essentiality_correlations.csv`: Pearson correlations between MTUS1 expression and Chronos dependency scores after the essentiality filter.
- `source_depmap_selected_candidates.csv`: selected MTUS1-associated dependency candidates.
- `depmap_mtus1_associated_essential_genes.csv`: source table used for Supplementary Table S of dependency candidates.
